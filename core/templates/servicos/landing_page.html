Página web que aparece em resposta ao clique em um resultado de busca de um mecanismo de pesquisa, é uma página que direciona o consumidor para aquilo que ele precisa.


É uma página de destino geralmente exibirá uma página de vendas direcionadas que é uma extensão lógica do anúncio, resultado de pesquisa ou link. As páginas de destino são usadas para geração de ‘leads’.


As ações que um visitante realiza em uma página de destino é o que determina a taxa de conversão de um anunciante. Uma página de destino pode ser parte de um microsite ou uma única página no site principal de uma organização.
